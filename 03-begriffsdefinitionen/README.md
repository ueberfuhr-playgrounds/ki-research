# Begriffsdefinitionen

> [!NOTE]
> Ziel ist eine Abgrenzung der Begriffe voneinander sowie deren Zusammenhang zu definieren.

## KI-Prompt

> Du bist IT-Trainer und vermittelst Nicht-Techies (Office-Nutzer) KI-Grundlagen. Dabei ist es wichtig, Begriffe korrekt zu definieren und in Zusammenhang zu bringen. Vor allem:
> - Modell
> - Agent
> - Extension (Gemini) bzw. Connectors (Claude)
> - RAG
> - MCP Server
> - Gems (Gemini) bzw. Projekte (Claude)
>
> Bringe diese Begriffe in Zusammenhang. (Wer nutzt was? Welche Multiplizität?) Stelle mir eine Auflistung kurzer prägnanter Fakten auf. Z.B. "Ein Agent nutzt ein Modell zum Denken und Extensions zum Handeln.", "Mehrere Agenten können dasselbe Modell nutzen." usw.

## Visualisierungs-Prompt

> Bekommst Du diese Begriffe visualisiert? Ich stelle mir vor, dass die Begriffe in einer Fläche angeordnet sind. Per Klick darauf rutscht alles etwas zusammen, und es öffnet sich ein Bereich. mit einer genauen Beschreibung, mit Beispiel und einer passenden Grafik.
>
> Vor allem bei den Extensions sind Icons für Outlook, Excel oder so sinnvoll. MCP und RAG als technische Begriffe sollten nicht im Vordergrund stehen, sondern lediglich im Text als Detail in Klammern stehen.
