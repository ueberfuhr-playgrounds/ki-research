# KI-Modellvergleich

> [!INFO]
> Ziel ist ein Überblick über die gängigsten KI-Modelle zum aktuellen Stand (Dezember 2025).

## KI-Prompt

> Ermittle die Vorzüge der Modelle GPT-5.1, Sonnet 4.5, Opus 4.5, Haiku 4.5, Gemini 3, o3, Grok-4. Ich benötige eine Einschätzung über Geschwindigkeit, Kontext-Fenster, Genauigkeit, Ausdauer bei langlaufenden Aktivitäten, Support für Programmierer, Support für Office-Nutzer, Multimedia-Fähigkeit, Kosten. Gerne auch in einer tabellarischen Darstellung mit den Kriterien als Zeilen und den Modellen als Spalten. Gerne auch in der letzten Zeile noch ein Fazit, wann welches Modell sinnvoll sein kann.
