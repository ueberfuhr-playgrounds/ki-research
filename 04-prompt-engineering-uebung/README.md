# Prompt Engineering Übung

> [!NOTE]
> Ziel ist das Verständnis für gute Prompts.

## KI-Prompt

> Ich habe neulich ein Video gesehen, in dem ein Vater seinem Kind den Auftrag gab, ihm aufzuschreiben, wie er ein Sandwich zubereitet. Er hat die Schritte dann durchgeführt, und zwar exakt so wie beschrieben. Das war recht komisch, weil es anfangs eben nicht funktionierte, und das Kind dann mehrfach seine Anleitung präzisieren musste.
>
> Ich stelle mir das nun auch als eine Übung für eine KI-Schulung zum Thema Prompt Engineering vor. Allerdings können wir in Präsenz und online kein Sandwich zubereiten. Welche Beispiele wären in diesem Kontext besser geeignet?
